# Image Gallery - React Project

This is a responsive **Image Gallery** web application developed using **React, HTML, and TailwindCSS**. The project displays a collection of images in a clean and user-friendly layout that adapts smoothly across different screen sizes.

The application is built using **core React concepts**, especially **Props**, to dynamically pass and display image data through reusable components.
